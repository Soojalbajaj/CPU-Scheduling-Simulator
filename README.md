# CPU-Scheduling-Simulator
A C++ Operating System project that simulates CPU scheduling algorithms (FCFS, SJF, Priority, Round Robin). The program runs on Ubuntu/Linux and displays waiting time, turnaround time, and Gantt chart.
### Usage Instructions
1. Compile the C++ file using g++.
2. Run the executable on Ubuntu/Linux.
3. Input number of processes, arrival times, burst times, and priorities.
4. Choose scheduling algorithm to see results and Gantt chart.
